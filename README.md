# 🌐 ThreatOrbit

> Interactive 3D globe mapping nation-state APT operations, attack arcs, and full intelligence profiles. Single HTML file. No install.

## Coverage

40 APT groups across 12 nation-states — Russia, China, North Korea, Iran, Vietnam, India, Pakistan, Turkey, Belarus, Israel, Lebanon, Palestine.

Notable actors: Fancy Bear · Sandworm · Cozy Bear · Turla · Volt Typhoon · Salt Typhoon · Double Dragon · Lazarus Group · BlueNoroff · Charming Kitten · OilRig · MuddyWater

---

## Features

- Directed attack arcs from origin to target nations
- Motivation filter — Espionage / Financial / Destructive / Mixed
- Click any country marker → all groups in that nation
- Full-text search across names, aliases, malware, CVEs, sectors
- Per-group dossier: TTPs, malware, CVEs, advisories, indicted members
- All MITRE ATT&CK IDs link to `attack.mitre.org`
- All CVEs link to NVD

---

## Usage

Download `index.html` and open it in any modern browser. No server, no dependencies, no API keys.
